**🎓 Sistema de Registro Académico - Colegio San José 
**
Este es un sistema de gestión escolar desarrollado en ASP.NET MVC diseñado para digitalizar la administración de estudiantes, materias y calificaciones.

✨ Funcionalidades Principales
Gestión de Estudiantes: CRUD completo para el registro y administración de la información de los alumnos.
Catálogo de Asignaturas: Administración de las materias disponibles en la institución.
Ingreso de Calificaciones: Módulo especializado para el registro de notas finales con validaciones.
Dashboard Analítico: Visualización estadística mediante gráficas. 
Persistencia Automatizada: Configuración de base de datos SQLite con auto-generación de esquema al iniciar la aplicación (Database.EnsureCreated).
